# LSudoku
一个简单的数独游戏应用
---------------
包含以下技术点：
* 数独棋盘的生成算法
* 横竖屏适配
* 屏幕旋转适配
* 简单主题框架
* 颜色选择器项目
* 通用型项目基础
* 动画的基本使用
* Preference基础使用
* Preference自定义
* View自定义
* Kotlin基本使用
* Drawable基本使用
* RecyclerView基本使用
* Sqlite基本使用
* Permissions基础使用
